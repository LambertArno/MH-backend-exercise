## React SSR

Server side-rendering is crucial for a whole lot of reasons.

Explain in your own words why it is crucial and how you would approach it.


## Explanation

### Pro's

* No need for loaders -> better UX
* The entire page can be crawled by seo robots, instead of only crawling a helmet or a soft-render
* Easy embeds (fixable by softrenders in CSR tho)

### Cons

* Higher level of expertise needed than CSR
* Higher server costs
